# Blueprint Smiles Website

Website for **Blueprint Smiles**, a doctor-led dental support organization serving the greater Atlanta area.

Live site: https://blueprint-smiles.github.io/

## Pages

| File | Description |
|---|---|
| `index.html` | Home page: hero, About Blueprint Smiles, Services, Why Choose Us, Benefits, Our Mission |
| `about.html` | About Us page with brand story and core values |
| `contact.html` | Contact page with info card and message form |

## SEO Files

| File | Description |
|---|---|
| `sitemap.xml` | Lists all pages for search engine crawlers |
| `robots.txt` | Allows crawling and points to the sitemap |
| `favicon.ico` | Browser tab icon |

## Deployment

This site is built with plain HTML/CSS (no build step required) and is hosted via **GitHub Pages**.

To deploy or update:
1. Push changes to the `main` branch of this repository.
2. In repo **Settings > Pages**, make sure the source is set to the `main` branch, root folder.
3. Changes go live automatically at `https://blueprint-smiles.github.io/`.

## Notes

- Replace the placeholder logo mark, phone number, email, and office hours in `contact.html` and the header/footer with real business details when available.
- If a custom domain (e.g. `blueprintsmiles.com`) is connected later, add a `CNAME` file to the repo root and update `sitemap.xml` and `robots.txt` URLs accordingly.
